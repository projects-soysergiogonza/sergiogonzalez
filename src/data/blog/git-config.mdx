<span style={{background: '#132362', paddingInline: '20px', width: 'fit-content'}}>**[Descargar Git](https://git-scm.com/downloads)**</span>

Es importante tener instalado <span style={{color: "#EA580C"}}>**`Git`**</span> en tu computadora para poder utilizarlo.\
En algunas computadoras basadas en Unix (Linux y Mac) ya viene instalado por defecto, pero en Windows es necesario descargarlo e instalarlo.
Apenas hemos instalado `Git`, estará disponible para todas las `terminales` del Sistema Operativo, así mismo si lo descargas manualmente en cualquier Sistema Operativo, también se va a instalar un Software llamado `GitBash`.

`GitBash` es una terminal que nos permite utilizar los comandos y funcionalidades similar a los que encontrarías en sistemas operativos como Linux o macOS. Si bien se puede instalar en los demás sistemas operativos, en Windows es donde más se aprovecha su uso, porque en los otros sistemas operativos ya se tiene una terminal nativa. Es importante mencionar que el uso de `GitBash` es opcional, ya que puedes utilizar la terminal de comandos de Windows para ejecutar los comandos de Git.

También se pueden instalar otras terminales de comandos diferentes a las terminales nativas de cada `SO`, a
continuación te dejo una lista de terminales que puedes instalar en tu computadora.
- **[Windows Terminal](https://aka.ms/terminal)**
- **[Hyper](https://hyper.is/)**
- **[Terminator](https://terminator-gtk3.readthedocs.io/en/latest/)**
- **[Alacritty](https://alacritty.org/)**

En linux y mac aparte de la terminal nativa, también recomiendo usar:
- **[iTerm2](https://iterm2.com/)**
- **[Warp (Mi Favorito)](https://www.warp.dev/)**

Ahora comencemos a usar la terminal y Git.

Cuando abrimos la terminal y queremos usar `Git` desde un principio es necesario identificarse con Git, para ello
debemos configurar nuestro nombre de usuario y nuestro correo electrónico, esto es importante, ya que cada
confirmación de cambios que realicemos en Git contendrá información sobre quién realizó cambios en el código.

Aquí te listo los comandos necesarios para realizar la configuración inicial de Git.

- **El primer paso es configurar el nombre de usuario.**

```bash:Example
  # Configurar el nombre de usuario
  git config --global user.name "Nombre de Usuario"
```

- **El segundo paso es configurar el correo electrónico.**

```bash
  # Configurar el correo electrónico
  git config --global user.email correo@correo.com
```

- **Recomiendo cambiar la rama principal de `master` a `main`.**

```bash
  # Cambiar el nombre de la rama principal
  git config --global init.defaultBranch main
```

- **Verificar la configuración realizada.**

```bash
  # Verificar la configuración realizada
  git config --list
```

Con esta configuración inicial ya podemos comenzar a utilizar Git en nuestro proyecto.
